SCPReader


@interface SCPReader : NSObject <SCPJSONDecodable>

Information about a reader that has been discovered or connected to the SDK. Note some of the properties are specific to a certain device type and are labeled as such. If a property does not call out a specific device then it is applicable to all device types.

See

https://stripe.com/docs/terminal/readers
  • ipAddress

    The IP address of the reader. (Verifone P400 only.)

    Declaration

    Objective-C

    @property (nonatomic, readonly, nullable) NSString *ipAddress;
  • locationId

    The location ID of the reader. (Verifone P400 only.)

    See

    https://stripe.com/docs/api/terminal/locations

    Declaration

    Objective-C

    @property (nonatomic, readonly, nullable) NSString *locationId;
  • status

    The networking status of the reader: either offline or online. Note that the Chipper 2X and the WisePad 3’s statuses will always be offline. (Verifone P400 only.)

    Declaration

    Objective-C

    @property (nonatomic, readonly) SCPReaderNetworkStatus status;

    Declaration

    @property (nonatomic, readonly, nullable) NSString *label;
  • batteryLevel

    The reader’s battery level, represented as a boxed float in the range [0, 1]. If the reader does not have a battery, or the battery level is unknown, this value is nil. (Chipper 2X and WisePad 3 only.)
